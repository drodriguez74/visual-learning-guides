§ 04 — How Similarity Is Measured
Cosine
cos(θ) = A·B / |A||B|
Measures the angle between two vectors, ignoring magnitude. Same direction = 1. Opposite = −1.
★ Best for NLP / embeddings
Euclidean
√Σ(aᵢ − bᵢ)²
Straight-line distance in space. Sensitive to magnitude — longer vectors appear further even if direction matches.
★ Good for image embeddings
Dot Product
Σ aᵢ · bᵢ
Magnitude × alignment. Faster to compute than cosine (no normalization). Used in ANN retrieval systems at scale.
★ Used in FAISS / ANN index
§ 05 — The RAG Pipeline
1
Ingest

Chunk Documents

Split source documents into overlapping chunks (~512 tokens each). Overlap preserves context across chunk boundaries.

2
Encode

Embed Each Chunk

Each chunk is passed through an embedding model (e.g. text-embedding-3) to produce a dense 1536-dimensional vector.

3
Store

Index in Vector DB

Vectors are stored and indexed in Pinecone, pgvector, Chroma, or Weaviate. ANN index enables sub-millisecond retrieval at scale.

4
Retrieve

Query → ANN Search → Top-K

User query is embedded with the same model, then an approximate nearest-neighbor search returns the top-K semantically similar chunks.

5
Generate

Inject & Generate

Retrieved chunks are injected into the LLM's context window as grounding evidence — dramatically reducing hallucination.
