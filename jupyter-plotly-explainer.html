Plotly Chart Types — Live & Interactive

Every chart below is real Plotly output — hover, zoom, and click the legend. Plotly renders directly in Jupyter notebooks as interactive HTML. No static images.

Line Chart — Trend Over Time
px.line
fig = px.line(df, x='month', y='revenue', color='product')
Bar Chart — Comparison
px.bar
fig = px.bar(df, x='region', y='sales', color='category', barmode='group')
Scatter Plot — Correlation
px.scatter
fig = px.scatter(df, x='spend', y='revenue', size='units', color='region')
Heatmap — Correlation Matrix
go.Heatmap
fig = go.Figure(go.Heatmap(z=corr_matrix, colorscale='Viridis'))
Sunburst — Hierarchical Data Breakdown
px.sunburst
Why Sunburst?
Visualizes nested categorical data — like region → product → SKU — in a single view. Click a segment to drill in.
fig = px.sunburst(
  df,
  path=['region','product','sku'],
  values='revenue'
)
Jupyter Magic Commands — Superpower Shortcuts
%timeit code
Benchmarks a line of code — runs it many times and reports average execution time.
%%time
Times an entire cell's execution. Essential for optimizing slow operations.
%matplotlib inline
Renders matplotlib charts inside the notebook (not needed for Plotly — it's automatic).
%who / %whos
Lists all variables in the current kernel namespace. Useful for debugging memory.
%%writefile file.py
Writes the cell's content to a Python file. Export code directly from a notebook.
%load_ext autoreload
Auto-reloads imported modules when they change — no kernel restart needed during dev.
